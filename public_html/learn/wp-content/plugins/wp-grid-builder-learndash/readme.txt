== Description ==

= WP Grid Builder - LearnDash =

This add-on mainly adds new blocks, for LearnDash plugin, to the card builder.
Easily add course progression, the number of steps, or the course price in your cards.
WP Grid Builder can natively display and filter your LearnDash courses, but with this add-on, you will be able to easily display course informations.

**Key Features:**

* Course progress bar block
* Course progress percent block
* Course steps counter block
* Course lessons number block
* Course user activity block
* Course status block
* Course price block
* 2 demo cards included

This add-ons also provides support for LearnDash Course Grid add-on.
In that way you will still be able to use content entered with it:

* Course ribbon block (LearnDash – Course Grid add-on)
* Course short description block (LearnDash – Course Grid add-on)
* Course custom button text block (LearnDash – Course Grid add-on)
* Youtube, Vimeo and Wistia support (LearnDash – Course Grid add-on)

== Frequently Asked Questions ==

WP Grid Builder - LearnDash add-on requires WP Grid Builder plugin v1.1.5 at minimum
